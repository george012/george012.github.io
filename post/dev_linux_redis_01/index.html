CentOS7.X yum方式安装最新版本Redis - Wed, Aug 15, 2018

目录：

  • 1. 更新yum源
  • 2. 安装redis-rpm源
  • 3. 安装Redis
  • 4. 开机自启Redis
  • 5. 设置redis.conf
  • 6. PS:

1. 更新yum源

yum update

2. 安装redis-rpm源

yum install http://rpms.famillecollet.com/enterprise/remi-release-7.rpm

3. 安装Redis

yum --enablerepo=remi install redis

4. 开机自启Redis

systemctl enable redis

5. 设置redis.conf

  • bind 127.0.0.1 改为 bind 0.0.0.0(可选)
    vim /etc/redis.conf
