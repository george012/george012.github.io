1. 开启systemctl服务

  • 在升级为wsl2最新版后执行

    echo -e "[boot]\nsystemd=true" | sudo tee -a /etc/wsl.conf

  • 安装完后再PowerShell中重启WSL2

    # 重启
wsl --shutdown

  • 在WSL Linux系统中执行下面命令，查看systemctl是否启动

ps --no-headers -o comm 1

2. PS:

  • 此操作基于（root）权限操作，如用普通用户可在命令行前加“sudo”
