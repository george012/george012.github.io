2. 内存虚拟盘

2.1. 临时挂载

mount -t tmpfs -o size=512G tmpfs /datas/plots_tmp

2.2. 永久挂载

vim /etc/fstab

tmpfs	/datas/plots_tmp	tmpfs	defaults,size=512G	0 0
