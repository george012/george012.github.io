1. 说明:

  • 记录常用命令用以备忘

2. Git操作

2.1. 将变化提交到暂存区

git add .

2.2. 提交并填写相关信息

git commit -m "release-Version-xxx"

2.3. 打tag

git tag x.x.x

2.4. 同步tag

git push --tags

3. pod trunk相关

3.1. 注册pod trunk

# Description 为描述
# xxx@xxx.com 为邮箱
# --tag       为标识
#--description  自我描述

pod trunk register xxx@xxx.com '--Tag' --description='Description'

3.2. 在项目根目录下，创建podspec文件

# PodName 为定义开源库名
pod spec create PodName

3.3. 查看注册信息

pod trunk me

3.4. 验证Pods库是否正确

# 验证线上
pod spec lint XXX.podspec
# 验证本地
pod lib lint --allow-warnings

--allow-warnings 指忽略警告

3.5. 上传至cocoaPods

pod trunk push --allow-warnings
