Ubuntu 14.0.4 phpMyAdmin 报错："缺少mcrypt扩展,请检查 PHP 配置"解决方案 - Wed, Jan 6, 2016

目录：

  • mcryp模块安装
  • mcrypt模块加载
    • 1. 开启mcrypt模块
    • 2. 查看apache2 配置文件
  • 重启apache2
  • PS:

mcryp模块安装

sudo apt-get install php5-mcrypt

mcrypt模块加载

1. 开启mcrypt模块

sudo php5enmod mcrypt

2. 查看apache2 配置文件

  • 进入目录："/etc/php5/apache2/conf.d/" 和 “/etc/php5/mods-available/” 检查是否有mcrypt.ini文件

重启apache2

sudo service apache2 restart
