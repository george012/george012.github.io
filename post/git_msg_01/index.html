git修改忽略文件“.gitignore”并立即生效 - Mon, Feb 20, 2017

1. 说明:

  • 有时候我们得修改忽略文件并立即生效,这样方便我们精细化管理git仓库

2. 清除缓存 记得 带“.”

git rm -r --cached .

3. 将变化提交到暂存区

git add .

4. 提交并填写相关信息

git commit -m "update .gitignore"

5. 可选同步到远程分支

git push

或者

git push origin master
